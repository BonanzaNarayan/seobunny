# **App Name**: MetaBoost

## Core Features:

- URL Analyzer: Fetch content from a given URL and analyze its existing metadata.
- Metadata Generation: Automatically generate SEO metadata, including title tags, meta descriptions, keywords, Open Graph tags, Twitter Card metadata, canonical URL, robots meta, and basic structured data.
- Metadata Optimization: An AI tool to rewrite and optimize existing metadata, using clearer titles, stronger descriptions, and keyword optimization, enhancing overall SEO performance.
- Metadata Display: Display the generated metadata in Next.js Metadata API format and standard HTML <head> metadata format.
- Metadata Copy: Provide a 'copy' button for each metadata section to quickly copy the content.
- Metadata Export: Allow users to export metadata in JSON and PDF formats.

## Style Guidelines:

- Primary color: Deep violet (#9400D3), suggesting sophistication.
- Background color: Light gray (#F0F0F0), creating a neutral backdrop.
- Accent color: Teal (#008080), offering a modern and contrasting visual interest.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to offer a neutral modern feel.
- Use minimalist icons for primary actions such as copy and export, ensuring clarity and ease of use.
- Incorporate subtle animations using Framer Motion for transitions and user feedback to enhance interactivity.
